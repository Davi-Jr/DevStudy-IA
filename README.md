## 🇧🇷 Sobre o Projeto

O **DevStudy IA** nasceu de uma dor real: **a falta de direção clara para quem quer evoluir como desenvolvedor.**

Somos bombardeados com cursos, tutoriais, frameworks e paths — mas ninguém diz **por onde começar** e **o que vem depois**.

Essa plataforma resolve isso gerando **roadmaps personalizados**, organizando seu aprendizado e acompanhando seu progresso com dados reais.

<div align="center">

> ### 💡 *"Pare de estudar tudo de uma vez. Estude o que importa, na ordem certa."*

</div>

### 🚀 Quick Start

```bash
# 1. Clone o repositório
git clone https://github.com/Davi-Jr/DevStudy-IA.git
cd DevStudy-IA/DevStudy-IA

# 2. Instale as dependências
npm install

# 3. Configure as variáveis de ambiente
cp .env.example .env

# 4. Inicie o servidor de desenvolvimento
npm run dev
```

> 🌐 Acesse **[http://localhost:5173](http://localhost:5173)**
